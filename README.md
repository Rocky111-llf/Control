# ## README

1 点灯以及串口通信采用hal库完成，源代码包含USER文件夹中的main.c文件、HARDWARE\LED文件夹中的led.c、led.h文件、HARDWARE\UART1文件夹中的uart.c\uart.h文件。LED文件夹中的源文件主要完成了GPIO时钟的使能以及GPIO的初始化，UART1文件夹中的源文件主要初始化了串口1，定义了接收数据中断函数以及数据处理回调函数（采用接受状态机实现），main.c主要利用自定义的串口通信协议，当接收到的数据中有编号为0x01的人员，两个LED灯以500ms间隔闪烁一次；当接收到的数据中有得分为0x02的人员，两个LED灯同时亮1s钟。

2 自定义的通讯协议在根目录下，定义了基地个人员得分情况的数据协议，可通过串口通信进行数据的传

输。
